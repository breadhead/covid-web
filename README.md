# coronairus-web
### prod https://defeatcovid.ru/

## Development
+ `git clone`
+ `npm i -g commitizen`
+ `yarn`
+ `yarn dev`

### if you want to use real api
+ `BACK_URL=https://api.defeatcovid.ru/ yarn dev`

```js
http://localhost:3001
```

### Checks
+ `yarn test` to run unit tests
+ `yarn lint:code` to check code-style
+ `yarn lint:style` to check style-style
